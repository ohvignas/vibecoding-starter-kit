# Dossier formateur

Ces documents sont pour **l'animateur** de la formation Vibe Coding — ils **ne sont jamais copiés** dans le projet d'un élève (le kit ne génère que le contenu de `templates/`).

- [`plan-de-cours.md`](plan-de-cours.md) — progression leçon par leçon, alignée sur les commandes et guides du kit, avec objectifs, durées et critères de réussite.

Le kit lui-même reste neutre : les projets générés ne parlent ni de « formation » ni d'« accompagnement ».
